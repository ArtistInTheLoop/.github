# CreatorsInTheLoop Games

In a nutshell: "We use AI to help us design games and humans then create them."

Our proprietary design pipeline stress-tests thousands of iterations to find the "perfect" game balance. But when it comes to the art and writing? That’s 100% human. We use technology to handle the heavy lifting of simulation so our small team can focus on what matters: intentional, handcrafted creativity.

It’s the power of a 50-person dev team, distilled into the vision of a few dedicated artists and writers.
